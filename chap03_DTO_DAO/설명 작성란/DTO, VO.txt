DTO - Data Transfer Object
// 데이터 전송을 목적으로 만들어진 객체
// 주로 외부 시스템과의 상호 작용, 원격 호출(RESTful API) 간의 데이터를 전달하기 위해 사용
// 데이터 전송을 위한 필드만 포함하고, 비즈니스 로직을 가지지 않는 경우가 많음 

VO - Value Object
// 읽기 전용 속성
// 비즈니스 도메인에서 중요한 역할 비즈니스 규칙을 나타내기 위해 사용
// 모델에서 일부만 사용이 되고, 특정 클래스의 상태를 나타내거나 변하지 않는 객체 생성

*RESTful 이란?
웹(HTTP) 통신을 위한 스타일 중 하나
